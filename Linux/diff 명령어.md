# diff 명령어

### 사용법

diff [option] [비교파일1] [비교파일2]

<br>

### option

| -c   | 두 파일간의 차이점 출력                                      |
| ---- | ------------------------------------------------------------ |
| -d   | 두 파일간의 차이점을 상세하게 출력                           |
| -r   | 두 디렉토리간의 차이점 출력, 서브디렉토리 까지 비교          |
| -i   | 대소문자의 차이 무시                                         |
| -w   | 모든 공백 차이무시                                           |
| -s   | 두 파일이 같을 때 알림                                       |
| -u   | 두 파일의 변경되는 부분과 변경되는 부분의 근처의 내용도 출력 |
