# find 명령어

### 사용법

find [-H] [-L] [-P] [-D debugopts] [-Olevel] [starting-point...] [expression]

명시한 경로 하위 경로 중 조건에 부합하는 파일을 찾는 명령어입니다.

<br>

### option

| 옵션    | 설명 |
| ------- | ---- |
| -name   |      |
| -type   |      |
| -mtime  |      |
| -delete |      |
| -not    |      |
