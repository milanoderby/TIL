# mkdir 명령어

### 사용법

mkdir [Option...] [생성할 디렉토리 경로]

디렉토리를 생성하는 명령어입니다.

<br>

### option

| 옵션 | 설명                                                         |
| ---- | ------------------------------------------------------------ |
| -p   | 명시한 디렉토리 경로 중간이 없을 경우, 해당 경로도 생성합니다. |
| -v   | 생성한 디렉토리를 출력합니다.                                |

<br>
